# titleblock
This project is to detect titleblock of 2D engineering drawings.

# Installation method
create file system as below.

1. build <br>
2. src 
    - public (clone https://github.com/captainys/public)
    - closed_region <br>
    - largest_column <br>
    - CMakeLists.txt <br>
